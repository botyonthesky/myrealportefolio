About Me

From an early age, I've always been very curious and have cultivated a wide range of interests.

From photography and travel to cinema and new technologies.

My passion for video games was born when I first discovered the world of Tomb Raider, which combined many of my interests into an experience in which I was free to explore and discover another world.


Games have evolved and so have I. And with the discovery of open worlds, such as Red Dead Redemption, I realized that there was a wonderful marriage between game development and that incomparable feeling of freedom.


After a somewhat atypical school and career path, I turned to development, particularly with Unreal Engine. And I discovered a world where anything was possible.


Since then, I've devoted all my time to learning and developing.


So here's my portfolio with some of the projects I've worked on.


I'm always happy to talk about new projects, ideas or just to chat, so don't hesitate to contact me!

RUN AWAY

This is my first game made with Unreal Engine.

I developed the entire game, from BluePrint programming to gameplay and mechanism design, focusing mainly on the gameplay loop to make the game as enjoyable as possible.


It's a simple game, in which you find yourself in an abandoned village (or almost) and you have to collect souls in order to escape.

CUB 3D

This is the CUB 3D game, created as a student project.

There were two of us, and I personally created all the UI. As well as the various animations, for weapons, doors and enemies.

This game is an adaptation of the famous Wolfeinstein 3d game, in which we were able to develop our creativity and rigor in the realization of a project.

The game is written entirely in C language.

SO LONG

This is my Solong game, created as a student project.

This game is made in C language with MiniLibX.

So long has enabled me to develop skills such as window management, event management, texture integration and the creation of small 2D animations.

This project forced me to be rigorous, to improve my C programming skills, and to use basic algorithms.
